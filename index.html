每本书对应一个 JSON 题库文件，可随时导入更新。

题库文件格式

每本书都分为“简答”“论述”“名词解释”三类。编辑 data/book-1.json、data/book-2.json、data/book-3.json 后刷新即可；每道题必须标注 category。

{
  "title": "第一本书",
  "categories": ["简答", "论述", "名词解释"],
  "items": [{ "id": "唯一编号", "category": "简答", "question": "题目", "answer": "答案" }]
}
